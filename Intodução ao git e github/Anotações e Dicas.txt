O que é git? 
Git é um sistema de controle de versão de arquivos.
Através deles podemos desenvolver projetos na qual 
diversas pessoas podem contribuir simultaneamente no 
mesmo, editando e criando novos arquivos e permitindo 
que os mesmos possam existir sem o risco de 
suas alterações serem sobrescritas...

*Alguns comandos Git
- Criar novo repositório - git init
- Verificar estado dos arquivos/diretórios - git status
-Adicionar um arquivo em específico - git add meu_arquivo.txt
- Adicionar um diretório em específico - git add meu_diretorio
- Adicionar todos os arquivos/diretórios - git add .
- Adicionar um arquivo que esta listado
no .gitignore (geral ou do repositório) - git add -f arquivo_no_gitignore.txt
 - Setar email - git config --global user.email seuemail@kddf.com.
- Setar usuário - git config --global user.name "SeuUsuario"
- Listar configurações - git config --list
- Clonar um repositório remoto já existente - git clone git@github.com:seurepositório.git
